# Naima Malik - CV

A professional and responsive CV website built with HTML and CSS, showcasing frontend development experience, skills, and education.

## Features

- **Responsive Design**: Two-column layout with sidebar and main content
- **Professional Profile**: Summary of background and career transition
- **Skills Section**: Display of technical skills including React, JavaScript, Node.js, and design tools
- **Education**: Formal training from Medieinstitutet, Technigo, and Brainstation
- **Work Experience**: Frontend developer roles at Emotional Fitness, IKEA, and New Default
- **References**: Professional contact information
- **Icons**: Font Awesome integration for visual appeal

## Technologies Used

- HTML5
- CSS3
- Font Awesome Icons (CDN)

## File Structure

```
CV/
```

## How to Use

Simply open `index.html` in a web browser to view the CV.

## Customization

Edit `index.html` to update personal information, skills, experience, or education.
Modify `style.css` to change colors, fonts, and layout styling.
